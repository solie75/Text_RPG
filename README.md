### Text RPG란?
레벨업과 전략적 아이템 관리를 통해 보스를 물리치는 텍스트 기반 로그라이크 RPG 게임 입니다.
![GameStructureDiagram](https://github.com/solie75/Text_RPG/blob/main/Game%20Structure%20Diagram.png)
게임을 시작하면 플레이어의 이름을 입력합니다.
